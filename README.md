## todo-list app

이 프로젝트는 Spring Boot, Kotlin, Querydsl을 사용하여 개발되는 to-do-list app 입니다. 프로젝트의 주요 기술 스택 및 구성은 다음과 같습니다:


1. Spring Boot (3.2.2)
2. Kotlin (1.9.22)
3. Querydsl (5.0.0)
4. Spring Data JPA
5. Spring Security
6. MySQL Database
7. Log4j2

## 프로젝트 구성 및 기능 설계 

- 사용자 인증 및 권한: Spring Security를 이용하여 사용자 인증 및 권한 부여를 구현
- sns 인증 회원가입 및 로그인
- to-do-list 추가, 수정, 조회, 삭제
- 오늘의 일기
